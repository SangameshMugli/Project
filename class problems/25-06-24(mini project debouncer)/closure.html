What is Closure?

closure concepts defines the below points mentioned :

  1. Inner function will be enclosed with the references to its surrounding state where it was declared.
  2. This references to surrounding state will allow function to access and manipulate those references even after the outer function scope has finished executing.
  3. Aclosure gives us access to outerfunctions scope from an inner function
  4. OR
  5. Closure: The innerfunction forms a closure. A closure is a function that retains access to its lexical scope, even when the function is executed outside that scope.
  6. lexical scope means that a function's scope is defined by its location within the source code, and nested functions have access to variables declared in their outer scopes.
